## About Me

보안은 시스템이 실제로 어떻게 동작하는지 이해하는 데서 출발한다고 생각합니다.<br>
`C` 기반 패킷 분석과 `Python` 기반 웹 보안 프로젝트를 통해 네트워크와 애플리케이션 계층의 데이터를 다루고 있으며, 임베디드 환경까지 관심 범위를 넓혀가고 있습니다.
